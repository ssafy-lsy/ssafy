

### 이중 for문

``` python
# 12345 출력 코드를 for문으로 감싸서 3번 반복 ---> 2중 for문

for i in range(3): # 3회 반복
    for j in range(1, 6): # 12345
        print(j, end = '')
    print() # 줄바꿈


# 다중 반복문 시작
arr = ['A', 'B', 'Q', 'T']

for i in range(4): # 4회 반복
    for j in range(3, -1, -1): # 역순 출력
        print(arr[j], end = '')
    print()

# 탐색 <---> 완전탐색
#
# 탐색 : 내가 원하는 것을 찾는것 (특정 값, 조건을 만족하는)
# 완전탐색(브루트포스 알고리즘) : 모든 경우를 정부 확인하는 것(재귀)

arr= [1, 4, 6, 1, 1, 9, 6]
# Q) 탐색해서 1이 몇개인지 counting 해보기
cnt = 0
for a in arr: # iterator 방식 순회
    if a == 1: cnt += 1

print(cnt)


# 2차원 리스트를 입력?
# 1차원 리스트를 입력 (4행 --> 4번 반복)

arr = [list(map(int, input().split())) for _ in range(4)]

for y in range(3, -1, -1): # 3부터 0까지 1씩 감소
    for x in range(3, -1, -1): # 3부터 0까지 1씩 감소
        print(arr[y][x], end = ' ')
    print() # 줄바꿈

```
